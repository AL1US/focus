# Focus

**Focus** — это минималистичная CLI-утилита для Unix подобных систем, которая помогает бороться с прокрастинацией. Она блокирует отвлекающие сайты на уровне системы (`/etc/hosts`), позволяя временно «разблокировать» их по расписанию через терминал.


## Установка 

1. Склонируйте репозиторий
```bash
git clone https://github.com/AL1US/focus
cd focus
```

2. Даём права на исполнение файлов
```bash
chmod +x focus install.sh uninstall.sh
```

3. Запустите установщик
```bash
sudo ./install.sh
```

4. Обновите конфиг оболочки
```bash
source ~/.zshrc
```

## Использование

| Команда | Описание |
| --- | --- |
| `focus add <сайт>` | Добавить сайт в черный список |
| `focus remove <сайт>` | Удалить сайт из списка |
| `focus list` | Показать текущий список блокировок |
| `focus unlock [мин]` | Разблокировать сайты на заданное время (по умолч. 5) |
| `focus block` | Принудительно заблокировать всё сейчас |
| `focus lang <ru/en>` | Сменить язык интерфейса |
| `focus help` | Справка по командам |


## Удаление

Чтобы полностью удалить утилиту из системы (включая очистку `hosts`, настроек и алиасов), перейдите в папку с клонированным репозиторием и выполните:

```bash
sudo ./uninstall.sh
```
